# Escrow-Payment-Contract

# 🚀 Escrow Payment Contract (Soroban - Stellar)

## 📌 Project Description
This project is a basic Escrow Payment Smart Contract built using Soroban SDK on the Stellar blockchain. It enables secure transactions between a buyer and a seller by holding funds in escrow until predefined conditions are met.

---

## ⚙️ What it does
- Locks funds from a buyer into a smart contract.
- Holds the funds securely until the buyer confirms delivery/service.
- Releases funds to the seller upon confirmation.
- Allows refund in case the transaction fails.

---

## ✨ Features
- 🔐 Secure escrow mechanism
- 👥 Buyer & Seller role-based authorization
- 💰 Fund locking and controlled release
- 🔄 Refund functionality
- 📦 Simple and modular contract design
- ⚡ Built using Soroban smart contract framework

---

## 🛠️ Functions Overview

| Function     | Description |
|-------------|------------|
| `init`      | Initializes escrow with buyer, seller, and amount |
| `deposit`   | Buyer deposits funds |
| `release`   | Buyer releases funds to seller |
| `refund`    | Seller refunds the buyer |
| `get_details` | Returns escrow details |
